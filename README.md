# Smart Medical Image Diagnosis System with Explainable AI

This project is a research-oriented multi-modal medical diagnosis system
designed as part of a thesis.

Primary modality:
- Chest X-ray (full research-grade implementation)

Secondary modalities (proof-of-concept):
- CT Scan
- MRI
- Pathology Reports

Core features:
- Disease detection and classification
- Explainable AI (Grad-CAM)
- Confidence and risk estimation
- Clinical decision support (non-prescriptive)

This system is designed as a research prototype, not a full production system.# Smart_Medical_Image_diagnosis_System_With_Explainable_AI
